Project Experience

浙江工业大学运河杯“诉求请愿”微信小程序

2019.06 – 2019.10

本项目内容是一个微信小程序，该小程序是希望提供一个平台让同学们公开自由的发布自己的诉求，并可以通过点赞、评论的形式，为诉求增加热度。后台将会把到达一定热度的诉求整理出来统一上报给校方，获得校方的反馈，由此搭建一个双向沟通的渠道，增强学生与学校之间的联系。本人在该项目中主要负责代码实现的部分，我使用了mpvue框架和vant、colorUI组件库，目前小程序已实现了基本功能，如发布诉求，评论，点赞，按诉求热度或时间排序等等，可落地使用，也获得校运河杯科技竞赛立项。

参与 IEEEVIS2020-MusicVis 研究课题

2020.01 - 2020.05

本项目是设计并实现了一个音乐可视化系统，该系统主要结合了空间分布视图和时间比较视图，可以帮助分析人员获得群体特征和详细解释。在该系统中，我们将音乐提取为带有时间戳的音符序列，并将音乐抽象为三个逐渐细化的信息(流派、乐器和音符)。 我参与的是时间比较视图部分，它主要显示音乐的音符序列，并进行容错相似序列检测，这可以帮助分析人员挖掘出音符分布的详细知识和微观解释。本人主要用D3.js这一可视化工具实现了前端音乐序列的对比。

参与话单分析研究项目

2020.11 – 2021.01

本项目旨在设计一款话单分析可视化系统，在展示页面中通过对话单数据的分析展示电话号码拥有者之间的联系，以此来判断他们之间的关系。在该系统中通过Neo4j存储话单数据，使用Vue + Element UI +Vis.js实现可视化界面。 本人负责前端部分，通过接口获取并处理数据，使用Vue + Element UI +Vis.js实现可视化界面。 ）
